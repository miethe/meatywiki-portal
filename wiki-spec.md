# MeatyWiki Portal — Wiki Spec

## Domain

MeatyWiki Portal is the Next.js 15 frontend for MeatyWiki, providing browser-based artifact browsing, note/URL capture, workflow monitoring, and compilation initiation through write-through calls to the Python backend.

## Topics

- Frontend surfaces: Inbox, Library, Artifact Detail, Quick Add, Workflow Status, Graph Explorer
- Component and design system patterns: Radix UI primitives, Tailwind, shadcn/ui composition
- API integration: SSE streams, cursor pagination, ETag caching, Portal backend contracts

- Hand-curate: naming, tagging, and routing conventions for this project's artifacts
